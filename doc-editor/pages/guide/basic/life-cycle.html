生命周期
生命周期用于在组件运行过程中的特定时间节点时，执行一段特定的逻辑。流程如下：
onMounted()
当组件第一次渲染完成时调用回调函数，你也可以在回调函数中返回一个函数，该函数将在组件销毁时调用。
import { onMounted } from '@viewfly/core'

function App() {
  onMounted(() => {
    // 组件挂载后调用
    return () => {
      // 组件销毁时调用
    }
  })

  return () => <div/>
}
Tsx/Jsx
onUpdated()
当组件每一次渲染完成时调用回调函数，你也可以在回调函数中返回一个函数，该函数将在下一次渲染完成时调用。
import { onUpdated } from '@viewfly/core'

function App() {
  onUpdated(() => {
    // 组件每一次渲染完成时调用
    return () => {
      // 组件下一次渲染完成时调用
    }
  })

  return () => <div/>
}
Tsx/Jsx
onPropsChanged()
当组件的 Props 发生变更时调用回调函数，并传入当前的和上一次的 Props。你也可以在回调函数中返回一个函数，该函数将在组件 Props 下一次变更时调用。
import { onPropsChanged } from '@viewfly/core'

function App(props) {
  onPropsChanged((newProps, oldProps) => {
    // 组件每一次渲染完成时调用
    return () => {
      // 组件下一次渲染完成时调用
    }
  })

  return () => <div/>
}
Tsx/Jsx
onUnmounted()
当组件销毁时调用回调函数。
import { onUnmounted } from '@viewfly/core'

function App() {
  onUnmounted(() => {
    // 组件销毁时调用
  })

  return () => <div/>
}
